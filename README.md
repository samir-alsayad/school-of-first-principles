# School of First Principles: The Forge (Base Branch)

This is the **Infrastructure Only** branch of the School of First Principles. It provides the core engine for building your own sovereign educational system.

## Included Components
- **`src/`**: The `school` CLI engine.
- **`rulesets/`**: The Sovereign Ruleset (Protocols and Identity).
- **`templates/`**: Pedagogical and Engineering scaffolding.

## Purpose
Use this branch as a clean slate for:
1. Importing specific tracks/modules from other repositories.
2. Building a new, private school from scratch.
3. Contributing to the core `school` CLI logic.

*To access the full curriculum, switch to the `curriculum` or `main` branches.*
