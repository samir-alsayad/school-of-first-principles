# Mission: Installation

## The Objective
Install the `gptme` driver in a clean environment.

## The Challenge
1.  Create a Virtual Environment (`.venv`).
2.  Activate it.
3.  Install `gptme-python` via pip.
4.  Run `gptme --version` to verify.

## Hints
*   Venv command: `python3 -m venv .venv`
*   Activation: `source .venv/bin/activate`

## Verification
Output should look like: `gptme version: 0.x.x`
